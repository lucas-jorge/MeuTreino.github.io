API Fitness Tracker

Sua solução completa para gerenciamento de exercícios e usuários

Sobre o Projeto

O API Fitness Tracker é uma poderosa API backend construída com ASP.NET Core, projetada para facilitar o gerenciamento de usuários e exercícios em aplicativos de fitness.

Principais Funcionalidades

Gerenciamento de Usuários

Crie, leia e atualize contas de usuário com facilidade.

Controle de Exercícios

Gerencie exercícios com operações completas de CRUD.

Autenticação Segura

Implementação de autenticação JWT para maior segurança.

Flexibilidade de Banco de Dados

Suporte para MySQL e banco de dados em memória.

Tecnologias Utilizadas

  • ASP.NET Core 3.1
  • Entity Framework Core
  • MySQL
  • JWT (JSON Web Tokens)
  • C#
